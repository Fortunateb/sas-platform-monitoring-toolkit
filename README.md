# sas-platform-monitoring-toolkit

A collection of generic, safe, and reusable scripts for SAS 9.4 and SAS Viya administration.

This repository showcases:
- SAS platform automation
- Monitoring and health-check tools
- Log management and cleanup scripts
- SAS 9 system checks
- SAS Viya REST API examples
- Python tools for admin tasks

All examples are generic and do **not** contain any confidential information.

### **sas9/**
Tools for SAS 9.4 administration:
- **log_management/** – log cleanup and scanning
- **metadata/** – metadata utilities (exporting, documenting)
- **monitoring/** – uptime and process checks

### **viya/**
Tools for SAS Viya:
- **cas_monitoring/** – CAS node health, performance checks
- **identities/** – user identity lookups
- **services/** – service status and restarts

 ### **python_tools/**
Python scripts using SAS Viya REST APIs. 

##  Purpose of This Repository

This repository is designed to:
- Demonstrate SAS platform administration skills  
- Provide useful tools for real-world enterprise environments  
- Support freelance projects and part-time consulting  
- Serve as a portfolio for engineering and DevOps-focused roles
